# Estructuras personalizadas

Guarda aquí todo lo que sí queremos versionar desde el servidor Linux o desde el
entorno local cuando construyamos contenido reutilizable.

Ejemplos recomendados:

- esquemáticos de WorldEdit (`.schem`)
- plantillas de dungeons
- regiones o layouts exportados
- notas de colocación y biome/theme

La idea es que el mundo vivo `server/world/` siga fuera de Git, pero las
estructuras reutilizables queden aquí.
