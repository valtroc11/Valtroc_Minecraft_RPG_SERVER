# Métricas y snapshots

Usa esta carpeta para guardar salidas resumidas que sí tenga sentido versionar:

- snapshots de balance
- tablas de pruebas de drops
- métricas semanales
- comparativas de rendimiento o spawn rates

No guardes aquí logs crudos gigantes ni dumps continuos del servidor. Para eso
conviene usar archivos externos o rotación local en la Linux.
