# Bases aprobadas para retexturizado

Esta es la regla operativa del proyecto para los items ilustrados del pack:

- una base aprobada por familia
- recolor puro sobre esa base
- sin geometria nueva
- sin adornos sinteticos pintados encima

## Interpretacion correcta

Si una familia necesita verse mas rica, primero se elige una mejor base usable del
pack medieval o del propio proyecto. No se compensa dibujando remaches, runas,
gemas o placas nuevas encima del PNG actual.
