# Progresion de Alquimia

La profesion **Alquimista** sube hoy de dos formas:

1. **Completar cocciones en Brewing Stand**
2. **Craftear Frascos de Experiencia por tier**

## XP base

- Cada coccion completada en un Brewing Stand da **18 XP de Alquimista**
- La XP necesaria para pasar de un nivel al siguiente es:

```text
180 x nivel actual
```

Ejemplo:

- nivel 1 -> 2: `180 XP`
- nivel 2 -> 3: `360 XP`
- nivel 3 -> 4: `540 XP`

## Ruta de desbloqueo de frascos

Los Frascos de Experiencia se desbloquean asi:

| Tier | Nivel de clase | Nivel de alquimista |
| --- | ---: | ---: |
| Madera | 1 | 1 |
| Piedra | 3 | 3 |
| Cobre | 5 | 5 |
| Bronce | 10 | 10 |
| Hierro | 18 | 16 |
| Plata | 30 | 26 |
| Diamante | 45 | 40 |
| Oricalco | 60 | 54 |
| Mithril | 78 | 72 |
| Legendario | 94 | 88 |

## Que hace cada frasco

Cada frasco:

- aumenta la **XP de clase**
- solo afecta mobs del **tier del frasco y el siguiente**
- solo permite **1 frasco activo a la vez**

## Camino recomendado

1. Sube Alquimista primero con cocciones normales
2. Revisa `/profesion alquimista`
3. Al llegar a un tier nuevo, lee el libro que te entrega el servidor
4. Craftea el frasco del tier para ganar XP adicional y mejorar tu farmeo de clase
